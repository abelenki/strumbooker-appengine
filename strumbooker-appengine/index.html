Strum What?

StrumBooker is a tool for creating and tracking guitar song sheets, called StrumSheets, and organizing those song sheets into collections, called StrumBooks.

Photo taken by Mr Paipai

How Does It Work?

It's quite simple actually.

1. Label the song with "Song:". Like this:

Song: Geeetar Song

2. Label the artist with "Artist:". Like this:

Artist: Geeetar Louie

3. Create chord charts with brackets. As many as you want. Like this:

[G:320033]
[D:xx0323]

4. Annotate chord changes in the lyrics with curly brackets. Like this:

Sing it, playa! 'till its time for a {G}chord change.
Sing it again, playa! 'till its time for another {D}chord change.

5. Put it all together, and you're ready to start strumming it out:

Song:Geeetar Song Artist:Geeetar Louie [G:320033] [D:xx0323] Sing it, playa! 'till its time for a {G}chord change. Sing it again, playa! 'till its time for another {D}chord change.
